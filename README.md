## 🎯 Scenario

An e-commerce brand is **losing customers during checkout**. The layout is cluttered and users are confused. The goal is to design a clean, logical structure for their landing page — starting from the skeleton, not the surface.

**The core problem:** The enemy is *Friction.*
> Friction is the gap between customer intent and site capability — the invisible resistance that turns a smooth journey into a slog.

---

## 🚀 Mission

- Design the **skeleton** of the application, focusing only on user flow and hierarchy
- Create **low-fidelity wireframes** for a Homepage
- Ensure clear **Call-to-Action (CTA) buttons** are placed strategically
- No color, no images — pure structural logic

---

## 🏛️ Information Architecture Applied

| Structure | Use Case |
|-----------|----------|
| **Hierarchical** | Visual importance & Gestalt theory |
| **Sequential** | Step-by-step checkout flow |
| **Matrix** | User-driven sorting and filtering |

**The 4 IA Systems:** Organization · Labeling · Navigation · Searching

---

## 👁️ Layout Patterns Used

### Z-Pattern
Best for minimalist pages with a single conversion goal.
- Logo → Nav links → Hero image diagonal → CTA button

### F-Pattern
Best for content-heavy pages and detailed product descriptions.
- Headline → Subheadline → Content blocks + vertical scan

## 📐 Wireframe Structure

```
┌────────────────────────────────────────┐
│  LOGO          Search Bar         🛒   │  ← Smart search, full-width
├────────────────────────────────────────┤
│                                        │
│  Hero Headline (benefit-driven)        │  ← Above the fold CTA
│  [ Get Started ]     [Hero Image]      │
│                                        │
├────────────────────────────────────────┤
│  Categories (visual icons, not links)  │  ← Shortcuts
├────────────────────────────────────────┤
│  [ Product ] [ Product ] [ Product ]   │  ← 3-column grid, 300px spacing
│  [ Product ] [ Product ] [ Product ]   │
├────────────────────────────────────────┤
│  [ Learn More ]                        │  ← Mid-page re-engagement CTA
├────────────────────────────────────────┤
│  [ Buy Now ]                           │  ← End-of-page conversion CTA
├────────────────────────────────────────┤
│  Footer                                │
└────────────────────────────────────────┘
```

---

## 📍 Strategic CTA Placement

| Zone | CTA | Purpose |
|------|-----|---------|
| Above the fold | Get Started | High-intent users |
| Mid-page | Learn More | Re-engagement after value content |
| End of page | Buy Now | Logic/trust-based conversion |
| Mobile sticky | Get Started | Always within thumb's reach |

---

## 🚫 Anti-Patterns Avoided

- No "Show All" hidden behind generic links — best-sellers must be visible
- No forced account creation at checkout
- No promo code box visible by default (stops users leaving for Google)
- No cluttered 5-column product grids (high cognitive load)
